# 처리량과 응답 시간
## 응답 시간
` 요청을 처리하는 데 걸리는 시간 `
### 과정
1. 서버 연결 : `TCP를 이용해 서버와 연결 `<br>
2. 데이터 전송 : `프로토콜에 따라 데이터를 전송`

### 서버 처리 시간
- 로직 수행
- DB 연동
- 외부 API 연동
- 응답 데이터 생성<br>

<strong>API 연동</strong>과 <strong>DB 연동</strong>에 시간이 많이 걸려,<br>
 응답 시간을 줄이기 위해선 <strong>API 연동</strong>과 <strong>DB 연동</strong>에 집중해야한다

 ## 처리량
 ` 단위 시간당 시스템이 처리하는 작업량 `

 ### 단위
 <strong>TPS</strong> (Transaction Per Second) : 초당 트랜잭션 수를 의미<br>
 <strong>RPS</strong> (Request per second) : 초당 요청 수를 의미
