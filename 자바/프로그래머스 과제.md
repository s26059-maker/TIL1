# Java 객체지향 핵심 정리

## 1. 클래스와 객체

###    개념

* **클래스(Class)**: 객체를 생성하기 위한 설계도
* **객체(Object)**: 클래스를 기반으로 생성된 인스턴스

```java
class Car {
    String color;
}
```

```java
Car car = new Car();
```

 `new` 키워드로 객체가 메모리에 생성됨


## 2. 클래스 다듬기 (구성 요소 정리)

###  기본 구성

* **필드(Field)**: 데이터
* **생성자(Constructor)**: 초기화
* **메서드(Method)**: 기능

```java
class Car {
    String color;

    Car(String color) { 
        this.color = color;
    }

    void drive() { 
        System.out.println(color + " 자동차 운전 중");
    }
}
```

### 캡슐화 (Encapsulation)

* 외부에서 직접 접근 제한 → 안정성 증가

```java
class Car {
    private String color;

    public void setColor(String color) {
        this.color = color;
    }

    public String getColor() {
        return color;
    }
}
```


## 3. 상속 (Inheritance)

###  개념

* 기존 클래스의 속성과 메서드를 재사용

```java
class Animal {
    void sound() {
        System.out.println("소리");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("멍멍");
    }
}
```

 `extends` 키워드 사용

###  특징

* 코드 재사용
* 계층 구조 형성
* 오버라이딩 가능

```java
@Override
void sound() {
    System.out.println("멍멍");
}
```


## 4. 인터페이스 & 다른 형태의 클래스

### 인터페이스 (Interface)

* 구현 없이 **기능의 틀만 정의**

```java
interface Animal {
    void sound();
}

class Dog implements Animal {
    public void sound() {
        System.out.println("멍멍");
    }
}
```

 `implements` 사용


### 추상 클래스 (Abstract Class)

* 일부 구현 + 일부 추상 메서드

```java
abstract class Animal {
    abstract void sound();

    void eat() {
        System.out.println("먹는 중");
    }
}
```



### 차이 정리
* 인터페이스는 "무엇을 할 수 있는지"만 정의하는 기능 명세
* 추상 클래스는 공통 기능을 일부 구현해놓은 기본 틀
* 인터페이스는 여러 개 구현 가능하지만, 추상 클래스는 하나만 상속 가능
* 인터페이스는 구조 설계용, 추상 클래스는 코드 재사용에 더 적합



## 5. 예외 처리 (Exception Handling)

### 기본 구조

```java
try {
    int result = 10 / 0;
} catch (Exception e) {
    System.out.println("에러 발생");
} finally {
    System.out.println("항상 실행");
}
```


###  핵심 키워드

* `try`: 예외 발생 가능 코드
* `catch`: 예외 처리
* `finally`: 항상 실행



### 예외 던지기

```java
void test() throws Exception {
    throw new Exception("에러 발생");
}
```
