# ERD(Entity Relationship Diagram)
### DB 테이블 설계도를 글과 표로 정리한 문서

| 테이블명  | 컬럼명         | 타입           | PK/FK |
| ----- | ----------- | ------------ | ----- |
| User  | id          | BIGINT       | PK    |
| User  | email       | VARCHAR(255) |       |
| User  | password    | VARCHAR(255) |       |
| Order | user_id     | BIGINT       | FK    |
| Order | total_price | INT          |       |
 
 주로 위와 비슷한 형식으로 쓴다 


 <strong>ERD에서 관계는 보통 `1:1`, `1:N` 이렇게 표현함</strong>

 ### 1:1
 - 필수
    - ──┼─────┼──
 - 선택
    -  ──┼─────○──

 ### 1:N
 - 필수
    - ─┼────────┼<
 - 선택
    - ──┼─────○┼<
