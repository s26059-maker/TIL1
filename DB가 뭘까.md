# 데이터베이스 핵심 요약

## 1. 자료와 정보
- **자료(Data)** : 가공되지 않은 사실
- **정보(Information)** : 자료를 가공하여 **의사결정에 유용한 형태**

---

## 2. 데이터베이스 정의
- **관련된 데이터들의 체계적인 집합**
- 조직에서 **공동으로 사용하기 위해 통합·관리되는 데이터**

---

## 3. 데이터베이스 특징
- **실시간 접근성** : 질의에 즉시 응답  
- **계속적인 변화** : 삽입·삭제·갱신으로 최신 상태 유지  
- **동시 공유** : 여러 사용자가 동시에 사용  
- **내용 기반 참조** : 데이터 위치가 아닌 **내용으로 검색**

---

## 4. 데이터베이스 시스템
- 데이터 저장·관리 및 정보 제공을 위한 **컴퓨터 기반 시스템**

---

## 5. 데이터베이스 종류
- **계층형(HDBMS)** : 트리 구조, 부모-자식 관계  
- **네트워크형(NDBMS)** : 노드 형태의 네트워크 구조  
- **관계형(RDBMS)** : 테이블 기반, 가장 보편적  
- **분산 DB** : 여러 컴퓨터에 분산 저장

---

## 6. 데이터베이스 설계 순서
1. **개념적 설계** : 전체 구조 개념화 (E-R 모델)  
2. **논리적 설계** : 데이터 모델로 변환, 스키마 정의  
3. **물리적 설계** : 실제 저장 구조 및 접근 경로 설계
